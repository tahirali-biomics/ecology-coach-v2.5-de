# Ecology Coach

![Version](https://img.shields.io/badge/version-v2.5_DE-0b5b4d)
![Status](https://img.shields.io/badge/status-active-1a7a68)

**Ecology Coach** is an interactive, AI-assisted course companion for university-level ecology. The German learning edition combines lecture-grounded content, adaptive tutoring, scientific reasoning, formative assessment, specialist vocabulary, research-writing support, learner progress, and course coordination.

> The Zenodo DOI badge and final citation should be added here after the Zenodo record has been published.

## Core features

- Twelve ecology modules and twenty-four in-depth lessons
- Adaptive daily lessons
- Course-grounded AI learning coach
- Ecological scenario simulations
- Scientific writing feedback
- Objective and written lesson assessment
- Specialist vocabulary and cloud synchronisation
- Voice input and read-aloud support
- Persistent learner progress
- Role-controlled course coordination
- Responsive desktop, tablet, and smartphone interface

## Academic purpose

The application is designed to help students connect ecological concepts with scientific reasoning, evidence evaluation, quantitative interpretation, and precise scientific communication. It emphasizes assumptions, alternative explanations, limitations, and the distinction between observed ecological patterns and the processes that may have generated them.

The AI components provide formative support. They do not replace instructor judgement, primary scientific literature, or independent scientific reasoning.

## Source and content boundary

This repository contains application source required for the public deployment. Raw lectures, books, exercises, solutions, student work, instructor pipelines, private credentials, and restricted institutional materials are not part of the public deployment bundle.

The deployed browser application receives compiled frontend assets only. AI-provider credentials and protected server-side processing remain outside the frontend bundle.
