# Ecology Coach 2.5 — German

A full-stack university ecology learning platform with 12 modules, 24 lessons, adaptive AI tutoring, scientific writing feedback, speech support, cloud vocabulary, course coordination, invitations and semester management.

## Quick start

```bash
nvm use 22
npm ci
cp .env.example .env.local
npm run dev -- --host 0.0.0.0
```

See `DEPLOYMENT.md` and `SUPABASE-SETUP.md`.
